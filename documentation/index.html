Orbit - Scifi Bootstrap 4 HTML Theme

This futuristic Scifi Theme based on Bootstrap 4 is set up with npm & webpack. It automatically compiles bootstrap & the custom SCSS files.

Based on the Orbit UI Kit (you can buy it if you need the Photoshop files): Orbit UI Kit on CreativeMarket
Permission to use the design for a commercial theme was granted by the author.

Refer to the Bootstrap 4 documentatioon for detailed usage information for all elements: Bootstrap 4 Documentation

Distributable files

You can find the compiled files in the dist folder. These files are the files you use to add the theme to your website.

  • index.html: You can open this file in a browser to see a demo website with many examples.
  • app.css: This file includes all the fancy new styles and also the Bootstrap styles, so you don't have to include Bootstrap separately. (How to include a CSS file)
  • app.js: This file includes some additional scripts and the Bootstrap scripts, so you don't have to include Bootstrap separately. (How to include a JS file)
  • Pro Font: The webfont (for numbers & special headlines) can be found in the folder "dist/assets/fonts".
  • Titillium: The webfont is added via Google Fonts.
  • Font Awesome: This webfont is included as the icon font.

Source files

The source files can be found in the src folder. These files are used if you want to customize the theme further. It requires some knowledge in HTML/CSS and the tools node.js & npm to work with those.

  • index.js: Includes custom scripts.
  • partials: Includes header, footer and meta sections which are used on all pages.
  • pages: Includes example pages and all Bootstrap components in the elements folder.
  • scss: Includes custom variables & styles.

To work with the source files you need node.js & npm installed. Then follow the instructions below.

If you are new to this, there are excellent tutorials online on how to set everything up on Linux, Windows or Mac. Just search for "Install node.js and npm on Linux/Windows/Mac".

Build setup

Open your console/terminal and use the following commands:

# install dependencies
 npm install

# serve with hot reload at localhost:8080
# this port is often occupied, but you can use another port with the command below
 npm run dev
# run with your own local domain & port (e.g. project-layout.local:5050)
# add your local domain to etc/hosts (e.g. project-layout.local)
# start with your desired port (e.g. 5050)
 npm run dev -- --open --port 5050

# build for production with minification
 npm run build

# build for production and view the bundle analyzer report
 npm run build --report
